# springboot-patient-api
Spring Boot REST API for managing patients with CRUD operations, DTOs, validation, exception handling, and Swagger documentation.
springboot-patient-api

A RESTful API built with Spring Boot for managing patient data. This project demonstrates CRUD operations, DTO-based data transfer, validation, exception handling, and Swagger documentation. It’s designed as a clean, maintainable backend project suitable for portfolio or learning purposes.

Features

✅ Create, Read, Update, Delete (CRUD) operations for Patient entity

✅ DTO-based data transfer to maintain security and hide sensitive fields

✅ Input validation using Jakarta Validation (@NotNull, @Email, @Size, etc.)

✅ Global exception handling with @ControllerAdvice

✅ Consistent API response format

✅ API documentation using Swagger / OpenAPI

✅ PostgreSQL database integration using Spring Data JPA

Tech Stack

Backend: Java, Spring Boot, Spring Data JPA

Database: PostgreSQL

Validation: Jakarta Validation (@Valid)

Mapping: DTOs with ModelMapper

API Docs: Swagger (springdoc-openapi)

Build Tool: Maven
